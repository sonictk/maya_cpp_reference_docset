Class Description

3D point with single-precision coordinates.

Constructors

SignatureParametersDescription
MFloatPoint()  

Default constructor. Returns a new MFloatPoint object, initialized to the origin.

MFloatPoint(src) src - MFloatPoint, MPoint, MFloatVector or MVector

Copy constructor. Returns a new MFloatPoint object with its x, y, z and w coords set to the same values as src. If src is a vector then the new MFloatPoint's w coordinate is set to 1.0.

MFloatPoint(seq) seq - sequence of two, three or four Floats

Returns a new MFloatPoint object whose x, y, z and w coordinates are set to the elements of seq. If the sequence contains fewer than four values w will be set to 1.0. If the sequence contains fewer than three values z will be set to 0.0.

y - Float

w - Float 		Returns a new MFloatPoint object with the specified x, y, z and w coordinates.

Sequence Support

An MFloatPoint is treated a sequence of four Float values: [x, y, z, w].

len() returns 4.

Indexing and element assignment are supported.

Deletion, concatenation, repetition and slicing are not supported.

Comparison Support

MFloatPoint == MFloatPoint Returns True if each component of the first point is exactly equal to the corresponding component of the second.
MFloatPoint != MFloatPoint Returns False is any component of the first point is not exactly equal to the corresponding component of the second.

All other comparison operators will raise a TypeError exception.
