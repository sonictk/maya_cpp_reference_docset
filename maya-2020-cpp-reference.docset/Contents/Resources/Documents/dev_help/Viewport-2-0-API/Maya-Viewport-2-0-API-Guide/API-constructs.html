API constructs

With a basic understanding of the rendering framework this next section provides details on the actual constructs exposed in the OpenMaya API. The focal point is the render item with details on its interface and the associated resources.

Aside from a few exceptions plug-in writers can find all constructs in the OpenMayaRender SDK module. All new classes related to rendering in the new hardware API have been grouped under the new MHWRender namespace.
