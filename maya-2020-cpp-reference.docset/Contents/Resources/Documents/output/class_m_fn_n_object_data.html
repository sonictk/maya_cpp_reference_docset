Detailed Description

function set for nCloth object data

Class for transferring N object data between connections

Examples:
testNobjectNode/testNobjectNode.cpp, testNpassiveNode/testNpassiveNode.cpp, testNsolverNode/testNsolverNode.cpp, and testNucleusNode/testNucleusNode.cpp.
